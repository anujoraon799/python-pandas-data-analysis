# Python Pandas Data Analysis

## Project Overview

This project demonstrates data cleaning, transformation, exploratory data analysis (EDA), and business-oriented problem solving using Python and Pandas.

The analysis was performed on the Penguin dataset to answer analytical questions and derive meaningful insights from the data.

- Python
- Pandas
- NumPy
- Seaborn
- Jupyter Notebook
